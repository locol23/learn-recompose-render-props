# learn-recompose-render-props

learn render props with recompose

## Tech Stack

- [React](https://reactjs.org/)
- [Parcel](https://parceljs.org/)
- [Prittier](https://prettier.io/)
- [ESLint](https://eslint.org/)
- [Recompose](https://github.com/acdlite/recompose)

## Prerequisites

- [Node.js](https://nodejs.org/en/)
- [Yarn](https://yarnpkg.com/en)

## Getting Started

### Clone the repository

```bash
git clone https://github.com/locol23/learn-recompose-render-props.git
```

### Install dependencies

```bash
cd learn-recompose-render-props
yarn

# if you use npm
rm yarn.lock
npm i
```

### Develop

```bash
yarn build

#if you use npm
npm run build
```

Go to [http://localhost:1234](http://localhost:1234)
